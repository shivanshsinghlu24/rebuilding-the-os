SmartShell

Overview
SmartShell is an intelligent command-line utility that combines command execution, system monitoring, and safety analysis.

Features
Natural language command support
Real-time system monitoring
Unsafe command detection
Lightweight and fast

How It Works
User input is parsed and converted into executable commands.
Commands are checked for safety before execution.
System stats are monitored using system-level APIs.

Installation
git clone https://github.com/shivanshsinghlu24/rebuilding-the-os
cd smartshell
pip install -r requirements.txt
python main.py

Usage
Type commands normally
Type monitor to see system stats
Type exit to quit
